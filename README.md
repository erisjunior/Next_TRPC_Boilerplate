# NextJs + TRPC Boilerplate

Boilerplate for projects using NextJs and TRPC with Prisma

## Getting Started

```bash
yarn
yarn dev
```

## Run Migrations

#### Development

```bash
yarn migration
```

#### Production

```bash
yarn migration-prod
```

## Used Technologies

- TypeScript
- React
- Next.js
- Tailwind CSS
- React Hook Form
- tRPC
- Zod
- Prisma
- PostgreSQL
- NextAuth
- UploadThing
- Resend
- T3 Env
